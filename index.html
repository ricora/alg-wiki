環境構築する

このページはHonkitを使っています。

くわしくはhonkitのドキュメントを読んで編集してください。

インストールする際の手順:

  1. npmとnodeとnpxを入れておく。わからなければnode.jsの環境構築をするを参照してください。

  2. 以下のコマンドを打つ

    npm init --yes
npm install honkit --save-dev

  3. ディレクトリに移動し、npm run serveを打って出てきたURLにアクセス

    cd alg-wiki && npm run serve

困ったときはJS PrimerがHonKitで書かれているのでそちらのGHとかを見るといいはず。

なにか問題があればtwitter@NAUS3Eに連絡、またはissueを立ててください。
